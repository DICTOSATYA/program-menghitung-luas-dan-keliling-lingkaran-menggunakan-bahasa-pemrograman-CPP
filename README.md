# program-menghitung-luas-dan-keliling-lingkaran-menggunakan-bahasa-pemrograman-CPP-

Repositori ini berisi program sederhana dalam bahasa C++/CPP yang digunakan untuk menghitung luas dan keliling sebuah lingkaran berdasarkan input jari-jari (radius) dari user. Project ini cocok buat latihan dasar pemrograman, khususnya buat yang lagi belajar operasi matematika, input/output, serta struktur dasar kode di C++.

Fitur Program
- Mengambil input jari-jari lingkaran dari user
- Menghitung luas lingkaran dan keliling lingkaran (2 program berbeda)
- Menampilkan hasil perhitungan ke layar
